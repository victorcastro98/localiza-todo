# Lista de Tarefas (To-Do List)

App de lista de tarefas que permite adicionar, excluir, visualizar e marcar tarefas como concluída.

## Available Scripts

- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run preview`
